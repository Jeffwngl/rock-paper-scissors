# 🪨📄✂️ Rock Paper Scissors Game

A simple Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer and see who wins! But the game is rigged.

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript

## 🧠 Game Logic

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
